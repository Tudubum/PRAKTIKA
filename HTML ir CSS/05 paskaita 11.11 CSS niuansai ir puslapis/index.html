Vaikinis elementas, gali tureti tik viena tevini elementa.

Paveldejimas

Paveldeti stilius gali tik vaikinius elementas is tevinio elemento. Kiatip nebus.
Paveldimi stiliai yra tie kurie keicia elemento isvaizda, ne busena. Pvz.: color, font-size, font-family...
Nepaveldimi stiliai yra tie, kurie keicia elemento busena. Pvz.: width. height, display, border, postition...
Net paveldeti stiliai, gali nebuti atvaizduoti del stiliu prioritetu.

Numatytieji stiliai

Numaytieji stiliai yra tag'ams automatiskai narsykles suteikti stiliai. Tarp skirtingu narsykliu, nusmatytieji stiliai gali minimalistiskai keistis ir sitaip sugadinti svetaines vaizda skirtingu naryskliu vartotojams.
Tam, kad nebutu problemu kuriant dizaina skirtingomis narsyklems yra naudojami Reset arba Normalize CSS'ai.
Reset - resetina visus stilius(istrina stilius).
Normalize - suvienodina visus stilius tarp narsykliu ir panaikina dali kenkejisku stiliu.

CSS prioritetai

  1. Inline Style - 1000
  2. ID - 100
  3. Class - 10
  4. Element - 1
  5. Css points(balai)
  6. CSS Order(eiliskumas CSS kode)
  7. Default styles
  8. Inheritance(paveldejimas)

Esant dviem vienodo visu prioritetu balo selektoriams, keiciant ta pati elemento stiliu, yra atsizvelgiama i CSS eiliskuma. Kuris Css stilius aprasytas veliau, tas ir bus priraikomas.

!important- Jeigu imanoma nenaudoti!!! !important prierasas prie stiliaus ignoruoja visus prioritetus ir suteikia ta stiliu ir niekas jo negali pakeisti, apart kito !important prieraso, kuris turi aukstesni prioriteta css.

Icons

Ikona yra mazas paveiksliukas. Zemiau pateiktos ikonu bibliotekos, prgindines. Norint naudoti reikia tureti ju nuoroda(biblioteka).
Tam kad i puslapi ikelti ikonas reikia naudoti i arba span tag'a. Nes ikonos neturi joms priskirto tag'o. i - suteiktti italic srifta, span - isskirti, pastilizuoti.
Ikonos stilizuojamos kaip tekstas, neprskiriami width/height, o priskiriame font-size. Nepriskiriame background-color, o priskiriame kaip color.
