CSS rasymo budai

Inline

Linijinis stilius yra elementui, kuri stilizuojame sukuriame style atributa ir jo viduje aprasome stilius. Neturi perpanaudojimo galimybes ir yra skirtas tik tai testavimui.

Internal - Vidinis

Vidinis stilius yra, kai HTML failo viduje, HEAD tag'o(gali buti ir body dalyje) dalyje susikuriame style tag'a. Ir tame style tag'e rasome CSS koda. Turi siek tiek perpanaudojimo galimybiu, nes turi selektorius. Ir dazniausiai naudojamas testavimui.

External - isorinis

Isorinis stilius yra rasomas atskrame CSS faile ir yra prijungiamas prie HTML failo naudojant tag'a. Turi visas reikalingas/norimas perpanaudojimo galimybes yra placiausiai naudojamas ir galutinio projekto variante, turetu likti tik sis stiliaus rasymo budas.

Basic Selectors

Element / Tag selektorius

Selektinami visi nurodyto tag'ai pagal nurodyta elementa.
Pvz: h2 - > parinks visus h2 elementus esancius HTML'e.

Class selektorius

Klase sukuriama kokiu tik nori logisku ir trumpu pavadinimu. CSS lange tam kad nurodytume klase, sugalvoto pavadinimo pradzioje parasome taska "." . HTML'e norint priskirti klase elementui, tam elementui suteikiame atributa "class" ir jam priskiriame sugalvota klases pavadinima be tasko.
Pvz.: CSS'e: .klasesVardas; o HTML'e: <p class="klasesVardas"> Klases turi daug su daug santyki. T.y vienas elementas gali turti daug klasiu ir viena klase gali buti priskirta daugeliui elementu. pvz: class="svarbusTekstas raudonasTekstas"> klases rasome per tarpeli, nereikia jokiu kabliataskiu.

ID selektorius

ID sukuriame kokiu tik norime logisku ir trumpu pavadinimu. CSS'e tam , kad nurodytume,jog kuriame ID - sugalvoto pavadinimopradzioje parasome groteles. HTML'e norint priskirti ID elementui, tam elementui suteikiame atributa "id" ir jam priskiriame sugalvota ID pavadinima be groteliu.
Pvz.: CSS'e #idPavadinimas; HTML'e: <p id="idPavadinimas">
ID turi viena su vienu santyi. Vienas elementas gali tureti TIK VIEN ID. Ir vienas id turetu buti priskirtas tik vienam elementui.

Colors

Names

Spalvos gali tureti vardus. Vardiniu spalvu yra 140. Norint priskirti elementui spalva, priskiri tos spalvos varda.
Pvz.:red; blue; green; lightgrey;

RGB - red green blue

RGB kiekvienos is siu triju spalvu kiekio galima ideti nuo 0 iki 255 ir sitaip isgauti bet kuria spalva.
Pvz.: rgb(0, 0, 0) - juoda. rgb(255, 0, 0) - raudonas;
RGBA - red green blue alfa. Alfa nurodo spalvos permatomuma ir jis gali buti nuo 0 iki 1 tukstanaja paklaida. Pvz.: rgba(0, 255, 0, 0) - nematoma(zalia); rgba(0, 255,0,1) - pilnai matoma (zalia)

Hex-hexadecimal

Hex - hexadecimal - sesioliktaine sistema. Kiekvienos is spalvos (RGB) galima nurodyti nuo 00 iki ff. Ir sitaip isgauti bet kuria spalva.
Skaiciavimas sesioliktaineje sistemoje: 00 01 02 03 04 05 06 07 08 09 0a 0b 0s 0d 0e 0f 10 11 12 13.... 19 1a 1b... 1f 20 21 22... 9f a0 a1 a2... a9 aa ab ac ad ae af b0.... ff.
pvz.: #000000 - juooda; #ff0000 - raudona; ....
Trumpiniai: #000 - juoda; #f00 - raudona;...
Spalvos matomumas su Hex: gale prisideda dar du sesioliktainiai simboliai, kur 00 - permatomas, ff - pilnai matomas.
Pvz.: #00ff0000 - nematoma (zalia); #00ff00ff - pilnai matoma(zalia);...
Trumpiniai: #0f00 - nematoma (zalia); #0f0f - pilnai matoma (zalia);....
