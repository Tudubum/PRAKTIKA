Stilizavimo tag'ai

  • b- bold - pastorina teksta
  • strong - pastorina teskta
  • i- italic - suteiia tekstui pasvirima
  • em - emphasize - suteikia tekstui pasvyrimo
  • sup - super - pakelia teksta i virsu ir ji sumazina(atrodo kaip kvadratu)
  • sub - nuleidzia teksta i apacia ir ji sumazina(atrodo kaip kvadratu)
  • span - isskiria kazkoki elementa ar jo dali, bet be stiliu (kaip ir visi linijiniai tagai)

Lenteles

Formos

Tag'ai

  • form - forma
  • input - ivestis
  • select - pasirinkimas is variantu
  • option - pasirinkimo variantai
  • textarea - didelio teksto kiekio ivestis
  • label - etikete prie kazkokios ivesties/pasirinkimo
  • optgroup - pasirinkimo variantu grupe
  • button - mygtukas
  • fieldset - formos laukas
  • legend - formos lauko pavadinimas

Atributai

  • type - Input tag'o tipas
  • for nurodo kuriam ivesties/parinkties laukui yra priskirta etikete
  • id - unikalus pavadinimas i kuri nurodo for atributas
  • name - įvesties lauko vardas, kuris bus perduotas back'ui kartu su tos įvesties reikšme
  • cols - stulpeliu kiekis textarea elemente
  • rows - eiluciu kiekis textarea elemente
  • placeholder - tekstas nusakantis ka rasyti ivesties lauke, kuris dingsta pradedant rasyti
  • value - ivesties reiksme, kuri perduodama i back'enda
  • label - priskiria colgroup'ui pavadinima
  • selected - numatytasis select tag'o viduje esantis pasirinkimo variantas
  • minlength - trumpiausia galima ivestis
  • maxlength - ilgiausia galima ivestis
  • step - zingsnis, kuriuo dideja/mazeja skaicius(desimtoji ar simtoji skaiciaus)
  • pattern - formatas nurodomas regex(regular expression)
  • checked - numatytasis radio arba checkbox pasirinkimas
