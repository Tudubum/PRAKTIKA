Tag'ų sąrašas

  • p - paragraph - paragrafas skirtas rašyti pastraipai
  • h1-h6 - Heading - antrašės skirtos pavadinimams ir tarpuspavyje skiriasi teksto dydžiu
  • img - image - pasveiksliukas, naudojamas su source ir alt, kitaip neveiks
  • a - hypertext anchor(link) - nuoroda į svetainę ir pan.
  • hr - Thematic Break (horizontal line) - sukuria eilutės lūžį su linija
  • br - line break - eilutės lūžis
  • ul - unordered list - nerikiuotas sąrašas
  • ol - ordered List - Rikiuotas sąrašas
  • list item - sąrašo elementas(skirttas tikiuotam arba nerikiuotam sąrašui)
  • meta - skirtas nurpdyti puslapio meta duomenis
  • tittle - skirtas nurodyti puslapio TAB'o pavadinimą
  • html lang - lietuviškas tekstas

Atributų sąrašas

  • lang - language - nurodo kokia kalba vyrauja tame tag'e(naudinga automatiniams skautyvuvams)
  • charset - charachter set - simboliu rinkinys, nurodo kokią simbolių biblioteką naudoti
  • src - source - Šaltinis nyrodantis failą
  • alt - alternative text - alternatyvus paveiksliuko neužkrovimo atveju
  • width - nurodo elemento plotį
  • height - nurodo elemento aukštį
  • href - Hyperlink reference - nuoroda į puslapį arba failą
  • target - nuoroda kur atidaryti nuorodą

Klaviatūros trumpinių sąrašas

  • CTRL S - išsaugoja failą
  • CTRL C - kopijuoja pažymėtą informaciją
  • CTRL X - iškerpa kopijuotą/iškirptą informaciją
  • CTRL V - įklijuoja kopijuotą/iškirptą informaciją
  • CTRL Z - atšaukia paskutinį veiksmą/us
  • CTRL F - Atdidaro paieškos lauką
  • TAB - tabliuoja pažymėtą kodą į dešinę
  • SHIFT TAB - tabliuoja pažymėtą kodą į kairę
  • CTRL A - pažymėti viską
  • CTRL / - komentuoja pažymėtas eilutes
  • CTRL Y - atšaukia CTRL Z veiksmą/us
  • ALT Z - word wrap
  • CTRL R - duoda pasirinkti ką atidaryti iš failų/aplankalų istorijos
  • alt + skaičių kombinacija - parašo skaičių kompinacijai priskirtą simbolį
  • CTRL SHIFT L - Pažymi visus vienoduus žodžius, taip pat galima juos keisti
  • CTRL SHIFT K - ištrina pažymėtą/as eilutę/es
  • CTRL SHIFT i/j | F12 - atidaro Developer Tools(naršyklėje)
  • CTRL D - pažymi tokį patį žodį einantį toliau teskte
  • END - nukels kursorių į eilutės galą
  • CTRL END - nukels kursorių į kodo galą
  • HM (home) - nukels kursprių į eilutės pradžią
  • CTRL HM (home) - nukels kursorių į kodo pradžią
  • SHIFT ALT arrow UP/DOWN -nukopijuoja pažymėtą/as eilutę/es aukštyn/žemyn
  • ! - sukuria HTML failo pradžią
  • CTRL ENTER - sukuria naują eilutę nelaužiant kitos jeigu kursorius yra per vidurį
  • CTRL SHIFT C - naršyklėje parenka INSPECTS element parinktį
  • ALT TAB - skirtas vaikščioti OS atidarytuose languose
  • SHIFT vaikščiojimo per kodą kombinacija, pažymi žodiužs per kuriuos kuraorius juda
  • CTRL SHIFT T - atidaro paskutinius uzdarytus tab'us
  • SHIFT ALT - keicia klaviaturos kalba
