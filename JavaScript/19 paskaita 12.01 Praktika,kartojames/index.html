Kartojimas

  1. Kintamieji(what, why, scope(apimtis), type, assign(priskirti))
    • string - zodinis kintamasis rasomas kabutese ''/ "" / ``;
    • number - skaitinis kintamasis
    • boolean/bool - loginis kintamasis (true/false)
    • array - masyvas
    • object - objektas

      • Objektas - Object - {raktas:reiksme, raktas1:reiksme1....} Masyvas - Array - [reiksme, reiksme, reiksme...] Tiek objektas, tiek masyvas yra konteineriniai kintamieji. Tai reiškia, kad jų viduje galima talpinti keletą reiškmių (gali būti skirtingų kitamųjų tipų).

  2. Operatoriai(darbas su kintamaisiais)

    • daugiau, maziau, daugiau arba lygu, maziau arba lygu
    • IR(&&), ARBA(||), NE(!)
    • tipas (typeof)(ar tai string, number, boolean, object, function

Nesting Salygos viduje, rasoma kita salyga.
