Events

Ivykiai - puslapyje vykstantys kazkokie veiksmai/ ivykiai(peles paspaudimas, klaviaturos paspaudimas, informacijos pazymejimas, scroll i virsu, apacia, formos pateikimas, failo ikelimas, atsisiuntimas is puslapio....).
Noredami stebeti(klausyti) ar nutinka koks ivykis, turime pradzioje nurodyti(selektinti) kurioje DOM dalyje tas ivykis gali nutikti.
Nurodzius(selektinus), kurioje dalyje klausomes ivykio - reikia aprasyti ivykio klausymasi. Tai padarome rasydami "addEventListener" metoda, kuris priima 2 parametrus: koksIvykis(peles paspaudimasm klaviaturos paspaudimas, peles uzejimas...), kaDarytiNutikusIvykiui.

Document Object Model

Dokumentas - puslapis - kuriame yra visa vartotojui atvaizduojama informacija(tekstas, paveiksliukas, forma, lentele...).
Objektas - kazkoks puslapio elementas(nuo html tag'o, iki ikonos tag'o)
Modelis - objektu struktura kazkuriame elemente(tekstas, paveiksliukas, forma, lentele...)

Noredami per JS selektinti kazkuria DOM dali, turime rasyti toki koda: "document.querySelector" arba "document.querySelectorAll". "QuerySelector" metodas priima 1 parametra - selekotriu(toki koki raseme CSS'e).
